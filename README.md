# Cyclistic Case Study

This project analyzes Cyclistic bike-share data to understand the behavioral differences between casual riders and annual members.

## Objective
Identify patterns in rider behavior and provide data-driven recommendations to help convert casual riders into annual members.

## Key Findings
- Members use the service more frequently  
- Casual riders take longer trips  
- Casual usage increases during summer, weekends, and afternoons
